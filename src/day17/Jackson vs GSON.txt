            e.printStackTrace();
        }


	}
}

Key Differences (Jackson vs Gson)
Feature			Jackson					Gson
Main class		ObjectMapper			Gson
Read JSON		readValue()				fromJson()
Type reference	TypeReference<List<T>>	TypeToken<List<T>>
File reading	File					FileReader

Use Jackson when:
Building enterprise applications
Using Spring / Spring Boot
Need high performance
Need advanced mapping control

Use Gson when:
Small projects
Simple JSON parsing
Android applications
Minimal configuration needed
