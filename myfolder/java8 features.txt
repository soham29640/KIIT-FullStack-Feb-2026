Map Examples
1. Convert each number to its square and print  	numbers.stream().map(n->n*n).forEach(n->System.out.println(n));
2. Convert each number to its square and collect in List  List<Integer> sqrNumbers= numbers.stream().map((num)->num*num).collect(Collectors.toList());
3. Multiply all numbers by 10 and collect in List	List<Integer> multipleOf10= numbers.stream().map((num)->num*10).collect(Collectors.toList());
4. display table of each number 

numbers.stream().map(n -> {
		    int fact = 1;

5. 4. Convert all month names to uppercase : 		months.stream().map(String::toUpperCase).forEach(System.out::println);

Filter Examples
1. Print even numbers								numbers.stream().filter(n->n%2==1).forEach(n->System.out.println(n));
2. Count how many numbers are > 3 : 				long count =numbers.stream().filter(n -> n > 3).count();
3. Print months starting with 'J' :  			months.stream().filter(m -> m.startsWith("J")).forEach(System.out::println);
4. Print months whose name length > 5 			months.stream().filter(m -> m.length() > 5).forEach(System.out::println);
5. Count months ending with 'y' : long count = 	months.stream().filter(m -> m.endsWith("y")).count();


Reduce Examples
1. Find the sum of all numbers			numbers.stream().reduce(0,(acc,num) -> acc + num);
